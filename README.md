Digital Literacy & Professional Presence Portfolio
Welcome to my Student Digital Ambassador project. This repository is a curated collection of resources, tools, and professional templates designed to promote digital literacy, cybersecurity awareness, and a professional online presence among college students.

 Project Overview
In 2026, being digitally literate means more than just using a computer; it involves critical thinking, ethical AI use, and proactive security. This project documents my journey in mastering these skills across five key modules.
